# Barebones i686-elf cross compiler

Built from instructions here: http://wiki.osdev.org/GCC_Cross-Compiler

| Package  | Version |
|----------|---------|
| binutils | 2.24    |
| gcc      | 4.9.1   |
| gmp      | 6.0.0a  |
| isl      | 0.13    |
| libiconv | 1.14    |
| mpc      | 1.0.2   |
| mpfr     | 3.1.2   |

Git clone, and set environmental variable: PATH="_install-dir_/cross/bin:$PATH"
